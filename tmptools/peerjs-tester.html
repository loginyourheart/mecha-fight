🔧 PeerJS 服务器测试工具

📖 使用说明

  1. 基础测试：输入服务器地址，点击"测试连接"检查服务器是否可达
  2. 完整测试：打开两个浏览器窗口（或两台设备），分别作为 Server 和 Client
  3. Server 端：输入房间号，点击"创建连接"，复制生成的 Peer ID
  4. Client 端：输入相同的房间号和 Server 的 Peer ID，点击"加入连接"
  5. 观察结果：如果 P2P 连接成功建立，说明服务器功能正常
