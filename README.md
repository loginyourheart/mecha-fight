# mecha-fight

ai web p2p game

## 像素风机甲对战游戏 v2.0.1

### 游戏介绍

一款像素风格的机甲对战游戏，支持单人模式和P2P联机对战。

### 在线演示

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-green)](https://loginyourheart.github.io/mecha-fight/)

在线体验：https://loginyourheart.github.io/mecha-fight/

### ⚠️ 重要提示

本游戏必须运行在HTTP服务器上！使用本地文件路径（file://）可能导致部分功能异常，推荐使用任意HTTP服务器（如 Python 的 http.server、VS Code 的 Live Server、GitHub Pages 等）运行。

### 操作说明

**单人模式（本地对战）**：
- 玩家1（红甲）：A/D 移动，W 跳跃，G 拳攻击，H 腿攻击，J 防御，Y 气功弹
- 玩家2（蓝甲）：←/→ 移动，↑ 跳跃，L 拳攻击，; 腿攻击，' 防御，P 气功弹

**联机模式**：
- A/D 移动，W 跳跃，J 拳攻击，K 腿攻击，I 气功弹，L 防御
- **S键**：攒气（按住站着不动就能攒气！）
- **超级火球**：100气时按 I

### 游戏特色

- 像素风格画面
- 实时P2P联机对战
- 流畅的战斗体验
- 多种攻击方式
- 防御反击机制
- 气值系统（普通火球/超级火球）
- 屏幕震动
- 击飞效果

### 技术说明

- 使用 PeerJS 实现 P2P WebRTC 联机
- 无需公网服务器，仅需信令服务器（使用公共 PeerJS 服务器）
- 适合局域网或具有NAT穿透条件的网络环境
- 位置同步使用 Lerp 平滑插值（系数 0.45）
- 游戏结束后可导出JSON格式完整日志
- **同步架构**：主机血量权威，气值本地控制（格斗游戏模式）
