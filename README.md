# 像素风机甲对战游戏 v1.1.1

## 游戏介绍
一款像素风格的机甲对战游戏，支持单人模式和P2P联机对战。

## 版本说明
v1.1.1 完善了网络同步功能：
- 防御状态实时同步
- 攻击动作平滑插值
- 位置同步优化（5px阈值 + Lerp平滑）
- 实时同步日志面板（按 ` 键显示/隐藏）
- 支持一键复制同步日志

## 如何运行

### 方法一：使用批处理文件（推荐）
1. 双击 `启动服务器.bat`
2. 在浏览器中访问 `http://localhost:8080`

### 方法二：手动启动
1. 打开命令行窗口
2. 进入本文件夹
3. 运行: `miniserve.exe . --port 8080 --index index.html`
4. 在浏览器中访问 `http://localhost:8080`

## 游戏操作

### 单人模式
- 玩家1（红色）：A/D移动，W跳跃，G拳击，H腿击，J防御，Y气功弹
- 玩家2（蓝色）：方向键移动，↑跳跃，L拳击，;腿击，'防御，P气功弹

### 联机模式
- 房主：点击"创建房间"获取房间号
- 游客：输入房间号加入游戏
- 双方使用相同键位控制自己的角色
- 主机显示红色标识，客户端显示蓝色标识

## 联机调试功能
- 底部实时同步日志面板（按 ` 键切换显示）
- 日志格式：[CLIENT/HOST] 时间 [SEND→/←RECV] 类型: 数据
- 点击"复制日志"按钮一键复制完整日志
- 点击"清空"按钮清空当前日志
- 游戏结束后可导出JSON格式完整日志

## 技术说明
- 使用 PeerJS 实现 P2P WebRTC 联机
- 无需公网服务器，仅需信令服务器（使用公共 PeerJS 服务器）
- 适合局域网或具有NAT穿透条件的网络环境
- 位置同步使用 Lerp 平滑插值，减少抖动
