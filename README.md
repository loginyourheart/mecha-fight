## 像素风机甲对战游戏 v2.0.0

### 游戏介绍

一款像素风格的机甲对战游戏，支持单人模式和P2P联机对战。

- **客机本地火球动画：
  - 客机自己发的火球本地能看到了！
  
- **气值显示和发送一致：
  - updateQiUI 用和 Math.round （和发送时一致，避免差1

  - rematch_accept 后也会判断是否双方都已接受并开始！

### 如何运行

#### 方法一：使用批处理文件（推荐）

1. 双击 `启动服务器.bat`
2. 在浏览器中访问 `http://localhost:8080`

#### 方法二：手动启动

1. 下载并解压 miniserve
2. 运行命令：`miniserve.exe .`
3. 在浏览器中访问显示的URL

### 操作说明

**单人模式（本地对战）：
- 玩家1（红甲）：A/D 移动，W 跳跃，G 拳攻击，H 腿攻击，J 防御，Y 气功弹
- 玩家2（蓝甲）：←/→ 移动，↑ 跳跃，L 拳攻击，; 腿攻击，' 防御，P 气功弹

**联机模式：**
- A/D 移动，W 跳跃，J 拳攻击，K 腿攻击，I 气功弹，L 防御
- **S键：攒气（按住，别移动就能攒气！
- **超级火球**：100气时按 I

### 游戏特色

- 像素风格画面

- 流畅的战斗体验
- 多种攻击方式

### 技术说明

- 使用 PeerJS 实现 P2P WebRTC 联机
- 无需公网服务器，仅需信令服务器（使用公共 PeerJS 服务器）
- 适合局域网或具有NAT穿透条件的网络环境
- 位置同步使用 Lerp 平滑插值（系数 0.45）
- 游戏结束后可导出JSON格式完整日志
- **同步架构**：主机血量，气值本地控制
